# Run GetStarted automation
1. Download Sahi framework from http://sahipro.com/
or direct link http://sahipro.com/static/builds/pro/install_sahi_pro_v750_20171222.jar
2. Istall Sahi Test Framework, after first run install the license  (license.data).
3. Put the file shipt_trian.sah into /sahi_pro/userdata/scripts folder
4. Start testrunner.sh with these parameters:
./testrunner.sh shipt_trial.sah https://google.com firefox
5. After test scipt finishes navigate to Sahi Logs to verify results.
